📰 Fake News Generator & Detector

A project that uses GPT-2 to generate realistic fake news articles, and a Logistic Regression classifier (trained on TF-IDF features) to detect if an article is REAL or FAKE. Fully implemented and tested in a Kaggle Notebook.

⚙️ How It Works

  1. User enters a news headline or prompt.
  2. GPT-2 generates a fake news article based on the prompt.
  3. The detector classifies the article as REAL or FAKE with confidence.

🧰 Technologies Used

  • Python 3
  • Hugging Face Transformers (GPT-2)
  • Scikit-learn (TF-IDF, Logistic Regression)
  • Kaggle Notebooks
  • GitHub Pages (for this site)

🚧 Challenges

  • Kaggle does not persist uploaded models; solved by retraining in notebook.
  • GPT-2 model loading delays; optimized with GPU and `max_length` adjustments.
